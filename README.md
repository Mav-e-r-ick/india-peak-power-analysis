## Project Overview

This project analyzing India's electricity peak demand problem using 9 Government of India datasets and quantitative models.

### Key Finding

- Renewables solved the capacity gap with a 42% renewable share.
- Renewables did not solve the timing gap because solar output is unavailable during the 8 PM peak.
- The recommended solution combines battery storage, carbon tax, and policy support.

## Project Structure

- `MB25026/Code` - notebooks and Python scripts used for data processing, analysis, and forecasting.
- `MB25026/Dataset` - source datasets tracked with Git LFS because several CSV files are larger than GitHub's standard file limit.
- `MB25026/Analysis_Outputs` - generated charts, summary tables, and recommendation outputs.
- `MB25026/PPT` - presentation decks.
- `MB25026/Report & Problem Statement` - report and problem statement documents.
- `MB25026/Project Report.pdf` - final project report.
- `Dataset` - top-level copy of the source datasets.
- Top-level weekly notebooks and PPT files - project milestone work and presentation material.
